# 💰 Expense Tracker App

A modern Flutter application for managing personal finances and tracking daily expenses.

## 🚀 Features
➕ Add Income & Expenses
📊 Category-wise expense tracking
📅 Date-wise transaction history
💰 Total balance calculation
📈 Beautiful UI dashboard
🔐 Firebase Authentication (if used)
☁️ Cloud data storage (if used)
🌙 Dark / Light theme support (if added)

🛠️ Tech Stack
Flutter (Frontend)
Dart
Firebase (Auth / Firestore) (if used)
GetX / Bloc (State Management)
REST API
